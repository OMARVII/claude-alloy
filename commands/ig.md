# ig — IGNITE Shorthand

`/ig` is an alias for `/ignite`. Use it when you want the full discipline mode: 6+ background agents firing in parallel, obsessive todos, review agents after every code change, and manual QA before declaring done.

Reach for this on non-trivial multi-file work, ambiguous specs, or anything labelled "important." Skip it for one-line fixes — the overhead isn't worth it there.

Note: the `hooks/ignite-detector.sh` hook also activates IGNITE whenever the user types `ig` or `ignite` in any prompt, so this command exists mainly as the documented entry point.

Run the `/ignite` command now.
