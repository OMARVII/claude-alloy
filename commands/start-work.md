# Start Work — Execute Plan

Begin working from the following plan: $ARGUMENTS

## Process:

1. **Load the plan**: If a plan file exists, read it. Otherwise, invoke @"carbon (agent)" first.
2. **Create todos**: Break the plan into atomic todo items
3. **Execute in order**: Follow dependency order from the plan
4. **Delegate**: Complex tasks → @"tungsten (agent)". Research → @"mercury (agent)" / @"graphene (agent)".
5. **Verify each step**: Run diagnostics and tests after each task
6. **Track progress**: Mark todos complete immediately

## Rules:
- Follow the plan exactly. Don't add scope.
- If the plan has gaps, consult @"prism (agent)" or @"quartz (agent)"
- If blocked, try 3 approaches before asking user
- Complete ALL plan items. No partial delivery.

Begin executing the plan now.

## Counterpart:
Use `/handoff` at the end of a session to create the context summary that this command consumes.
